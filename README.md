# 📝 备忘录工具

一个简洁美观、功能完整的纯前端备忘录应用，无需服务器即可使用。

## ✨ 功能特性

- 🎨 **美观界面**：现代化的渐变背景和毛玻璃效果
- 🗂️ **分类管理**：工作、个人、学习、其他四个分类
- 🔍 **实时搜索**：即时搜索备忘录内容
- 🎨 **多彩主题**：5种不同的背景颜色可选
- 📱 **响应式设计**：完美适配手机和电脑
- 💾 **本地存储**：数据保存在浏览器本地，永不丢失
- 📤 **数据导入导出**：支持JSON格式的备份和恢复
- ⌨️ **键盘快捷键**：支持Ctrl+N新建、Ctrl+F搜索等快捷键
- 👀 **视图切换**：网格视图和列表视图自由切换

## 🚀 使用方法

### 1. 直接打开
由于这是一个纯前端应用，你可以直接双击 `index.html` 文件在浏览器中打开使用。

### 2. 本地运行
如果你想通过本地服务器运行：

```bash
# 使用Python
python -m http.server 8000

# 使用Node.js
npx serve .

# 使用PHP
php -S localhost:8000
```

然后访问 `http://localhost:8000`

## 🎯 操作指南

### 新建备忘录
- 点击"新建备忘录"按钮
- 或按快捷键 `Ctrl+N` (Windows) / `Cmd+N` (Mac)

### 编辑备忘录
- 点击每个备忘录卡片上的编辑图标
- 修改内容后点击保存

### 删除备忘录
- 点击备忘录上的删除图标
- 确认删除后不可恢复

### 搜索备忘录
- 在顶部搜索框输入关键词
- 支持按标题和内容搜索
- 按 `Ctrl+F` 快速聚焦搜索框

### 分类筛选
- 使用下拉菜单选择分类
- 支持"全部分类"、"工作"、"个人"、"学习"、"其他"

### 切换视图
- 点击"切换视图"按钮在网格和列表视图间切换

### 数据备份
- 点击"导出"按钮下载所有备忘录为JSON文件
- 点击"导入"按钮选择之前备份的JSON文件恢复数据

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` / `Cmd+N` | 新建备忘录 |
| `Ctrl+F` / `Cmd+F` | 聚焦搜索框 |
| `Esc` | 关闭模态框 |

## 🔧 自定义

### 修改颜色主题
编辑 `styles.css` 中的颜色变量：

```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    /* 添加更多自定义颜色 */
}
```

### 添加新分类
编辑 `script.js` 中的分类配置：

```javascript
const categories = {
    work: '工作',
    personal: '个人',
    study: '学习',
    other: '其他',
    // 添加新分类
    newCategory: '新分类'
};
```

### 调整备忘录颜色
编辑 `index.html` 中的颜色选择器：

```html
<input type="radio" name="color" value="#你的颜色" id="color6">
<label for="color6" style="background-color: #你的颜色;"></label>
```
